Computed Properties

Computed properties in Vue 3 are values that are derived from one or more data properties. They are defined in the `computed` option and provide a way to perform calculations or manipulations on data in a declarative manner. In this example:

Base Price: The base price is a computed property that displays the product's original price. It is directly linked to the "Product Price" input field and updates automatically as the user changes the input value.

Discount: The discount is another computed property that shows the discount applied to the product. Similar to the base price, it is linked to the "Product Discount" input field and updates based on user input.

Calculate Total Button: This button triggers the `calculateTotal` method when clicked. The method recalculates the computed properties, including the total price, based on the current values of "Product Price" and "Product Discount" inputs.

Total Price: The total price is a computed property that represents the final price of the product after applying the discount. It updates automatically whenever the base price or discount changes, ensuring that the total price is always accurate.
