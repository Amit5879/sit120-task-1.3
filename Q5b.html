a. Binding HTML class [v-bind:class]

The concept of binding HTML classes using v-bind:class in Vue 3 allows you to dynamically apply or remove CSS classes to HTML elements based on data properties or conditions. In this example:

  • The highlighted class is bound to a <div> element.
  • When the isHighlighted data property is true, the highlighted class is applied, resulting in a yellow background.
  • Clicking the "Toggle Highlight" button toggles the value of isHighlighted, demonstrating how class binding dynamically changes the element's appearance.

b. Binding Inline Styles [v-bind:style]

The concept of binding inline styles using v-bind:style in Vue 3 enables you to apply custom CSS styles directly to HTML elements based on data properties. In this example:

  • The <p> element has inline styles bound to the customStyle data property.
  • customStyle defines custom CSS properties, such as color and fontSize.
  • As a result, the paragraph's text color is set to blue, and the font size is 18 pixels.
