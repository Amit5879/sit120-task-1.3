Vue 3 Watchers Concept

Watchers:

Watchers are a Vue 3 concept that allows you to perform custom logic when a specific data property changes. They are defined in the `watch` option of a Vue component. Watchers observe changes in one or more data properties and execute a function when the observed data changes. In this example, we have a watcher that observes changes in the `userInput` data property. When `userInput` changes, the watcher calculates the doubled value and updates the `doubledValue` data property accordingly.
