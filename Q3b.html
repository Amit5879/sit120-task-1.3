Q3: Reactivity Fundamentals in Vue 3

a. ref()

Vue 3 provides the ref() function, which is used to create a reactive reference to a value.

you to define reactive data that can be used in templates and methods. In this example, the counter is created as a reactive reference.

b. <script setup>

The <script setup> syntax is a feature of Vue 3 that simplifies the structure of single-file components. It allows you to define data, methods, and computed properties in a more concise and readable way. In this example, the message is defined using <script setup>.

Message: {{ message }}
